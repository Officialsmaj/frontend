# SMAJ Portfolio

Personal portfolio website for SALEH MALA AJIMI (SMAJ), showcasing a junior frontend developer profile, skills, projects, and experience.

## Features
- Single-page layout with section navigation
- Hero, About, Skills, Projects, Experience, and Contact sections
- Project modal and inline project details
- Responsive layout and animated visual elements

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Font Awesome (CDN)

## Project Structure
- `index.html` Main page content
- `styles.css` Styling and layout
- `script.js` Interactions and animations
- `images/` Media assets

## Usage
1. Open `index.html` in a browser.
2. Edit content directly in `index.html` and styling in `styles.css`.

## Customize
- Update images in `images/`
- Replace project placeholders in the Projects section
- Update the CV file referenced in the hero section
